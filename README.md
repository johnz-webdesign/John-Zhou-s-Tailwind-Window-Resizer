### **Keyboard Shortcuts**
Use `Alt + Shift + [Number]` to snap to breakpoints:

| Shortcut          | Window Size  |
|------------------|-------------|
| `Alt + Shift + 1` | `640px (sm)` |
| `Alt + Shift + 2` | `1024px (lg)` |
| `Alt + Shift + 3` | `1280px (xl)` |
| `Alt + Shift + 4` | `1536px (2xl)` |

### **Overlay**
- The overlay shows the **current window size**.
- It updates **automatically** when resizing.
